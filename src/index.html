FORM ER (DUM)ᵏ

Documentation

A programming language based on brainfuck presented on a "løbeseddel" from 15 April 2020. The "løbeseddel" can be found here.
Instructions
  • FORM ER DUM:
    increment the data pointer (to point the next cell to the right).
  • FORM ER DUM DUM:
    decrement the data pointer (to point the next cell to the left).
  • FORM ER DUM DUM DUM:
    increment (increase by one) the byte at the data pointer.
  • FORM ER DUM DUM DUM DUM:
    decrement (decrease by one) the byte at the data pointer.
  • FORM ER DUM DUM DUM DUM DUM:
    output the byte at the data pointer.
  • FORM ER DUM DUM DUM DUM DUM DUM:
    accept one byte of input, storing its value in the byte at the data pointer.
  • FORM ER DUM DUM DUM DUM DUM DUM DUM:
    if the byte at the data pointer is zero, then instead of moving the instruction pointer forward to the next command, jump it forward to the command after the matching FORM ER DUM DUM DUM DUM DUM DUM DUM DUM command.
  • FORM ER DUM DUM DUM DUM DUM DUM DUM DUM:
    if the byte at the data pointer is nonzero, then instead of moving the instruction pointer forward to the next command, jump it back to the command after the matching FORM ER DUM DUM DUM DUM DUM DUM DUM command.
